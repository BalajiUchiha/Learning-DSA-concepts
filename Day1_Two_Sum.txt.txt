Step 2: Brute-Force Approach (Understand First):

Try all pairs (i, j) and check if they add up to target.
Python:

def twoSum(nums, target):
    for i in range(len(nums)):
        for j in range(i + 1, len(nums)):
            if nums[i] + nums[j] == target:
                return [i, j]

Space Complexity: O(1)
This works but is slow when n becomes large.

Step 3: Optimized Approach (Using HashMap)
Instead of checking all pairs, we use a hash map to remember numbers weâ€™ve seen.
Logic:
â€¢	Loop through each number.
â€¢	For each number num, find its complement â†’ target - num.
â€¢	If complement exists in the hash map â†’ we found our pair.
â€¢	Otherwise, store num with its index in the hash map.

Python Code:

def twoSum(nums, target):
    seen = {}  # value -> index
    for i, num in enumerate(nums):
        complement = target - num
        if complement in seen:
            return [seen[complement], i]
        seen[num] = i
JS Code
function twoSum(nums, target) {
  const seen = new Map();
  for (let i = 0; i < nums.length; i++) {
    const complement = target - nums[i];
    if (seen.has(complement)) {
      return [seen.get(complement), i];
    }
    seen.set(nums[i], i);
  }
}
